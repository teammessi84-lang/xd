🔥 Cómo configurar Firebase (gratis)

Paso 1 — Entra a console.firebase.google.com con tu cuenta Google → Crear un proyecto → ponle nombre (ej: "asistencia2do") → sin Analytics está bien.

Paso 2 — En el menú izquierdo → Realtime Database → Crear base de datos → elige ubicación (Estados Unidos está bien) → Modo de prueba → Aceptar.

Paso 3 — Regresa al inicio del proyecto → clic en el ícono </> → registra app web → ponle nombre → copia el bloque que dice const firebaseConfig = { ... }

Paso 4 — Abre tu index.html → busca la sección marcada con 🔧 → reemplaza los 7 valores de firebaseConfig con los tuyos (apiKey, authDomain, etc.).

Paso 5 — Guarda y sube a GitHub. Abre en cualquier celular → todos verán la asistencia al instante. ✅
